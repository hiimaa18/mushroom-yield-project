# Random Forest Interpretation

Feature importance scores indicate which environmental variables contributed most strongly to prediction decisions.

Features with higher importance values influenced the model more frequently across decision trees.

Because Random Forest captures nonlinear relationships, feature importance reflects predictive usefulness rather than positive or negative direction.

The most important feature should be investigated further in future experiments and data collection efforts.

# Random Forest vs Linear Regression

Linear Regression remains easier to explain to farm stakeholders because coefficients directly indicate relationships between variables and yield.

Random Forest can capture nonlinear relationships and interactions automatically, but it is less interpretable.

If Random Forest produces substantially better test metrics, the additional complexity may be justified.

If performance improvement is small, Linear Regression may remain preferable because of its simplicity and transparency.
